Documentation

Welcome to the documentation for Taj's Graphical Overhaul (TGO). This comprehensive guide will help you install, configure, and get the most out of the visual enhancements provided by the mod.

This documentation assumes you're familiar with installing mods for Satisfactory using the Satisfactory Mod Manager (SMM).

System Requirements

Before installing TGO, ensure your system meets the following requirements:

Component Minimum Recommended
GPU GTX 1060 6GB RTX 2060 or better
VRAM 6GB 8GB+
CPU i5-6600 i7-8700 or better

Performance may vary depending on your factory size and complexity.

Installation Guide

Using Mod Manager (Recommended)

  1. Open Satisfactory Mod Manager
  2. Search for "Taj's Graphical Overhaul"
  3. Click the download button
  4. Wait for installation to complete
  5. Launch the game

Manual Installation

[Game Directory]/FactoryGame/Mods/TajGraphicalOverhaul/

Remember to back up your configuration file before making changes!

Frequently Asked Questions

Will this mod impact my game performance?

TGO is designed to be as performance-friendly as possible, but some impact is expected. Use the configuration options to find the right balance for your system.

Is this mod compatible with other visual mods?

TGO is generally compatible with other mods, but other visual enhancement mods might conflict. Check the compatibility section for details.

Can I use this mod in multiplayer?

Yes! TGO is client-side only and can be used in multiplayer without requiring other players to have it installed.

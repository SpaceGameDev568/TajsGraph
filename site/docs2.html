Documentation

Welcome to the documentation for Taj's Graphical Overhaul (TGO). This comprehensive guide will help you install, configure, and get the most out of the visual enhancements provided by the mod.

This documentation assumes you're familiar with installing mods for Satisfactory using the Satisfactory Mod Manager (SMM).

System Requirements

Before installing TGO, ensure your system meets the following requirements:

Component Minimum Recommended
GPU GTX 1060 6GB RTX 2060 or better
VRAM 6GB 8GB+
CPU i5-6600 / Ryzen 5 1600 i7-8700 / Ryzen 7 2700X
RAM 16GB 32GB

Performance may vary depending on your factory size and complexity. These requirements assume a medium-sized factory.

Installation Guide

Using Satisfactory Mod Manager

  1. Open Satisfactory Mod Manager
  2. Search for "Taj's Graphical Overhaul"
  3. Click the download button
  4. Wait for installation to complete
  5. Launch the game

Manual Installation

Manual installation is not recommended but can be done by following these steps:

1. Download the mod from ficsit.app
2. Extract the contents to [Game Directory]/FactoryGame/Mods/
3. Ensure the folder is named exactly "TajGraphicalOverhaul"

Enhanced Lighting

TGO implements several improvements to Satisfactory's lighting system:

Global Illumination

Enhanced bounce lighting creates more realistic indoor and outdoor environments. This affects:

  • Interior factory spaces
  • Underground areas
  • Shaded outdoor areas

Frequently Asked Questions

Will this mod impact my game performance?

TGO is designed to be as performance-friendly as possible, but some impact is expected. Use the configuration options to find the right balance for your system.

Is this mod compatible with other visual mods?

TGO is generally compatible with other mods, but other visual enhancement mods might conflict. Check the compatibility section for details.

Can I use this mod in multiplayer?

Yes! TGO is client-side only and can be used in multiplayer without requiring other players to have it installed.

Performance Settings

TGO includes several performance presets and customization options:

Performance Preset

Optimized for maximum FPS while maintaining essential visual improvements.

{
  "shadowQuality": "medium",
  "reflections": "low",
  "volumetrics": "disabled"
}

Quality Preset

Maximum visual fidelity for high-end systems.

{
  "shadowQuality": "ultra",
  "reflections": "high",
  "volumetrics": "enabled"
}

Configuration File

The main configuration file can be found at:

[Game Directory]/FactoryGame/Mods/TajGraphicalOverhaul/config.json

Remember to back up your configuration file before making changes!
